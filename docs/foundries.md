# Foundry list

Love & Logic's curated list of type foundries, plus Google Fonts and Fontshare.

Tiers reflect Love & Logic's own ranking from the bookmarks:
- **best**: top-tier foundries to reach for first
- **okay**: solid second tier
- **loose**: ungrouped from her root folder, treat as okay
- **notgood**: kept for completeness, only suggest if no better match exists

Types:
- **foundry**: an actual type foundry
- **reseller**: marketplace or aggregator (Fontshare, Future Fonts, Type Network etc.)
- **free**: free/libre source

Total: 222 entries.

## Default search order

Search **best** first. If nothing close matches, broaden to **okay** + **loose**. Only mention **notgood** entries if explicitly asked or as a last resort. For free/budget briefs, prefer entries where `type` is `free`.
